1    ============================================= 
2     worldlib -- library for country code lookup 
3    ============================================= 
4    
5    To use the ``worldlib`` library, import the database which reads a canned,
6    pre-generated set of code mappings:: 
7     
8        >>> from worldlib.database import Database 
9        >>> db = Database() 
10    
11   You can look up a country code:: 
12    
13       >>> print(db.lookup_code('it')) 
14       Italy 
15    
16   Country codes are case-insensitive:: 
17    
18       >>> print(db.lookup_code('It')) 
19       Italy 
20    
21   You can find all matches for a particular string, which allows you for example
22   to implement a reverse look up.   The matches are returned sorted in
23   alphabetical order.  As with code look ups, the match string is case
24   insensitive:: 
25    
26       >>> for code, country in db.find_matches('United'): 
27       ...     print(code, 'is', country) 
28       ae is United Arab Emirates (the) 
29       gb is United Kingdom of Great Britain and Northern Ireland (the) 
30       tz is Tanzania, United Republic of 
31       uk is United Kingdom (common practice) 
32       um is United States Minor Outlying Islands (the) 
33       us is United States of America (the) 
34    
35   You can iterate through all the codes:: 
36    
37       >>> for code in db: 
38       ...     print(code) 
39       ac 
40       ... 
41       zw
