150  <p>IMPORTANT NOTE: THIS MODULE IS DEPRECATED IN PYGAME 2.2</p>
151  <p>In older pygame versions before pygame 2, <a class="tooltip reference internal" href="event.html#module-pygame.event" title=""><code class="xref py py-mod docutils literal notranslate"><span class="pre">pygame.event</span></code><span class="tooltip-content">pygame module for interacting with events and queues</span></a> was not well
152  suited for posting events from different threads. This module served as a
153  replacement (with less features) for multithreaded use. Now, the usage of this
154  module is highly discouraged in favour of use of the main <a class="tooltip reference internal" href="event.html#module-pygame.event" title=""><code class="xref py py-mod docutils literal notranslate"><span class="pre">pygame.event</span></code><span class="tooltip-content">pygame module for interacting with events and queues</span></a>
155  module. This module will be removed in a future pygame version.</p>
156  <p>Below, the legacy docs of the module is provided</p>

177  <dl class="py function definition">
178  <dt class="sig sig-object py title" id="pygame.fastevent.pump">
179  <span class="sig-prename descclassname"><span class="pre">pygame.fastevent.</span></span><span class="sig-name descname"><span class="pre">pump</span></span><span class="sig-paren">(</span><span class="sig-paren">)</span><a class="headerlink" href="#pygame.fastevent.pump" title="Permalink to this definition">¶</a></dt>
180  <dd><div class="line-block">
181  <div class="line"><span class="summaryline">internally process pygame event handlers</span></div>
182  <div class="line"><span class="signature">pump() -&gt; None</span></div>
183  </div>
184  <p>For each frame of your game, you will need to make some sort of call to the
185  event queue. This ensures your program can internally interact with the rest
186  of the operating system.</p>
187  <p>This function is not necessary if your program is consistently processing
188  events on the queue through the other <a class="tooltip reference internal" href="#module-pygame.fastevent" title=""><code class="xref py py-mod docutils literal notranslate"><span class="pre">pygame.fastevent</span></code><span class="tooltip-content">pygame module for interacting with events and queues</span></a> functions.</p>
189  <p>There are important things that must be dealt with internally in the event
190  queue. The main window may need to be repainted or respond to the system. If
191  you fail to make a call to the event queue for too long, the system may
192  decide your program has locked up.</p>
193  </dd></dl>

195  <dl class="py function definition">
196  <dt class="sig sig-object py title" id="pygame.fastevent.wait">
197  <span class="sig-prename descclassname"><span class="pre">pygame.fastevent.</span></span><span class="sig-name descname"><span class="pre">wait</span></span><span class="sig-paren">(</span><span class="sig-paren">)</span><a class="headerlink" href="#pygame.fastevent.wait" title="Permalink to this definition">¶</a></dt>
198  <dd><div class="line-block">
199  <div class="line"><span class="summaryline">wait for an event</span></div>
200  <div class="line"><span class="signature">wait() -&gt; Event</span></div>
201  </div>
202  <p>Returns the current event on the queue. If there are no messages
203  waiting on the queue, this will not return until one is available.
204  Sometimes it is important to use this wait to get events from the queue,
205  it will allow your application to idle when the user isn't doing anything
206  with it.</p>
207  </dd></dl>

230  <dl class="py function definition">
231  <dt class="sig sig-object py title" id="pygame.fastevent.post">
232  <span class="sig-prename descclassname"><span class="pre">pygame.fastevent.</span></span><span class="sig-name descname"><span class="pre">post</span></span><span class="sig-paren">(</span><span class="sig-paren">)</span><a class="headerlink" href="#pygame.fastevent.post" title="Permalink to this definition">¶</a></dt>
233  <dd><div class="line-block">
234  <div class="line"><span class="summaryline">place an event on the queue</span></div>
235  <div class="line"><span class="signature">post(Event) -&gt; None</span></div>
236  </div>
237  <p>This will post your own event objects onto the event queue. You can post
238  any event type you want, but some care must be taken. For example, if you
239  post a MOUSEBUTTONDOWN event to the queue, it is likely any code receiving
240  the event will expect the standard MOUSEBUTTONDOWN attributes to be
241  available, like 'pos' and 'button'.</p>
242  <p>Because pygame.fastevent.post() may have to wait for the queue to empty,
243  you can get into a dead lock if you try to append an event on to a full
244  queue from the thread that processes events. For that reason I do not
245  recommend using this function in the main thread of an SDL program.</p>
246  </dd></dl>
