142  <dl class="c function">
143  <dt class="sig sig-object c" id="c.pgRect_FromObject">
144  <span class="n"><span class="pre">SDL_Rect</span></span><span class="w"> </span><span class="p"><span class="pre">*</span></span><span class="sig-name descname"><span class="n"><span class="pre">pgRect_FromObject</span></span></span><span class="sig-paren">(</span><span class="n"><span class="pre">PyObject</span></span><span class="w"> </span><span class="p"><span class="pre">*</span></span><span class="n"><span class="pre">obj</span></span>, <span class="n"><span class="pre">SDL_Rect</span></span><span class="w"> </span><span class="p"><span class="pre">*</span></span><span class="n"><span class="pre">temp</span></span><span class="sig-paren">)</span><a class="headerlink" href="#c.pgRect_FromObject" title="Permalink to this definition">¶</a><br /></dt>
145  <dd><p>Translate a Python rectangle representation as a Pygame <code class="xref c c-type docutils literal notranslate"><span class="pre">SDL_Rect</span></code>.
146  A rectangle can be a length 4 sequence integers (x, y, w, h),
147  or a length 2 sequence of position (x, y) and size (w, h),
148  or a length 1 tuple containing a rectangle representation,
149  or have a method <em>rect</em> that returns a rectangle.
150  Pass a pointer to a locally declared <code class="xref c c-type docutils literal notranslate"><span class="pre">SDL_Rect</span></code> as <em>temp</em>.
151  Do not rely on this being filled in; use the function's return value instead.
152  On success, return a pointer to a <code class="xref c c-type docutils literal notranslate"><span class="pre">SDL_Rect</span></code> representation
153  of the rectangle, else return <em>NULL</em>.
154  No Python exceptions are raised.</p>
155  </dd></dl>
