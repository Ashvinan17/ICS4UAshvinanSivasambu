1    File Path Function Arguments
2    ============================
3    
4    A pygame function or method which takes a file path argument will accept
5    either a Unicode or a byte (8-bit or ASCII character) string.
6    Unicode strings are translated to Python's default filesystem encoding,
7    as returned by sys.getfilesystemencoding().  A Unicode code point
8    above U+FFFF (``\uFFFF``) can be coded directly with a 32-bit escape sequences
9    (``\Uxxxxxxxx``), even for Python interpreters built with an UCS-2
10   (16-bit character) Unicode type.  Byte strings are passed
11   to the operating system unchanged.
12   
13   Null characters (``\x00``) are not permitted in the path, raising an exception.
14   An exception is also raised if an Unicode file path cannot be encoded.
15   How UTF-16 surrogate codes are handled is Python-interpreter-dependent.
16   Use UTF-32 code points and 32-bit escape sequences instead.
17   The exception types are function-dependent.
