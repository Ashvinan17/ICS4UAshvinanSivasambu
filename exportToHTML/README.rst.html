2    world -- Print mappings between country names and DNS country codes 
3    =================================================================== 
4    
5    This package provides a mapping between top-level domain names and their two
6    letter ISO_ 3166_ country codes.  This script also knows about non-geographic,
7    generic, USA-centric, historical, common usage, and reserved top-level
8    domains.
9    
10   Both a command line script called ``world`` and a library called ``worldlib``
11   are available.  The latter can be imported into your Python code for whatever
12   application you want.
13   
14   
15   Requirements 
16   ============ 
17   
18   At a minimum, Python 3.5 is required.
19   
20   
21   Documentation 
22   ============= 
23   
24   A `simple guide`_ to using the library is available within this package.  The
25   manual is also available `online`_.  A manpage provides information on how to
26   use the ``world`` command line script.
