44   .. function:: get_finger
45   
46      | :sl:`get information about an active finger`
47      | :sg:`get_finger(touchid, index) -> int`
48   
49      :param int touchid: The touch device id.
50      :param int index: The index of the finger to return
51                        information about, between 0 and the
52                        :func:`number of active fingers <pygame._sdl2.touch.get_num_fingers()>`.
53   
54      Return a dict for the finger ``index`` active on ``touchid``.
55      The dict contains these keys:
56   
57      ::
58   
59         id         the id of the finger (an integer).
60         x          the normalized x position of the finger, between 0 and 1.
61         y          the normalized y position of the finger, between 0 and 1.
62         pressure   the amount of pressure applied by the finger, between 0 and 1.
