250  <p>These examples should help get you started with pygame. Here is a brief rundown
251  of what you get. The source code for these examples is in the public domain.
252  Feel free to use for your own projects.</p>
253  <p>There are several ways to run the examples. First they can be run as
254  stand-alone programs. Second they can be imported and their <code class="docutils literal notranslate"><span class="pre">main()</span></code> methods
255  called (see below). Finally, the easiest way is to use the python -m option:</p>
256  <div class="highlight-default notranslate"><div class="highlight"><pre><span></span><span class="n">python</span> <span class="o">-</span><span class="n">m</span> <span class="n">pygame</span><span class="o">.</span><span class="n">examples</span><span class="o">.&lt;</span><span class="n">example</span> <span class="n">name</span><span class="o">&gt;</span> <span class="o">&lt;</span><span class="n">example</span> <span class="n">arguments</span><span class="o">&gt;</span>
257  </pre></div>
258  </div>
259  <p>eg:</p>
260  <div class="highlight-default notranslate"><div class="highlight"><pre><span></span><span class="n">python</span> <span class="o">-</span><span class="n">m</span> <span class="n">pygame</span><span class="o">.</span><span class="n">examples</span><span class="o">.</span><span class="n">scaletest</span> <span class="n">someimage</span><span class="o">.</span><span class="n">png</span>
261  </pre></div>
262  </div>
263  <p>Resources such as images and sounds for the examples are found in the
264  pygame/examples/data subdirectory.</p>
265  <p>You can find where the example files are installed by using the following
266  commands inside the python interpreter.</p>
267  <div class="highlight-default notranslate"><div class="highlight"><pre><span></span><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">pygame.examples.scaletest</span>
268  <span class="gp">&gt;&gt;&gt; </span><span class="n">pygame</span><span class="o">.</span><span class="n">examples</span><span class="o">.</span><span class="n">scaletest</span><span class="o">.</span><span class="vm">__file__</span>
269  <span class="go">&#39;/usr/lib/python2.6/site-packages/pygame/examples/scaletest.py&#39;</span>
270  </pre></div>
271  </div>
272  <p>On each OS and version of Python the location will be slightly different.
273  For example on Windows it might be in 'C:/Python26/Lib/site-packages/pygame/examples/'
274  On Mac OS X it might be in '/Library/Frameworks/Python.framework/Versions/2.6/lib/python2.6/site-packages/pygame/examples/'</p>
275  <p>You can also run the examples in the python interpreter by calling each modules main() function.</p>
276  <div class="highlight-default notranslate"><div class="highlight"><pre><span></span><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">pygame.examples.scaletest</span>
277  <span class="gp">&gt;&gt;&gt; </span><span class="n">pygame</span><span class="o">.</span><span class="n">examples</span><span class="o">.</span><span class="n">scaletest</span><span class="o">.</span><span class="n">main</span><span class="p">()</span>
278  </pre></div>
279  </div>
280  <p>We're always on the lookout for more examples and/or example requests. Code
281  like this is probably the best way to start getting involved with python
282  gaming.</p>
283  <p>examples as a package is new to pygame 1.9.0. But most of the examples came with
284  pygame much earlier.</p>

642  function to shift the image on the display surface.
643  A clip rectangle protects a margin area. If called as a function,
644  the example accepts an optional image file path. If run as a program it
645  takes an optional file path command line argument. If no file is provided a
646  default image file is used.</p>
647  <p>When running click on a black triangle to move one pixel in the direction
648  the triangle points. Or use the arrow keys. Close the window or press
649  <code class="docutils literal notranslate"><span class="pre">ESC</span></code> to quit.</p>
650  </dd></dl>
